
# Al-Aswani — Bilingual Editorial Travel Site

A premium, handcrafted-feeling travel site for Al-Aswani (Netherlands → Europe). Arabic-first (RTL) with full English (LTR) parity, separate routes per section, WhatsApp as the primary booking CTA, and all imagery/branding generated by me.

## Information architecture

Routes (TanStack Start, file-based):
- `/` — Home (the long editorial landing described in the brief)
- `/services` — Tourist trips, Transportation, Arab drivers, Hotel booking (deep dive)
- `/packages` — Postcard-cluster of curated European trips
- `/about` — Story of Al-Aswani, values, team feel
- `/contact` — WhatsApp CTA, hours, service areas, FAQ

Shared shell:
- Header with logo, nav, language switcher (AR ⇄ EN), small WhatsApp pill
- Footer styled as the closing page of a travel journal
- Floating WhatsApp button on all pages

## Bilingual & RTL system

- Default language: Arabic. Switcher persists choice in `localStorage`.
- `<html lang dir>` set at the root route from a `LanguageProvider` (React context). All layout uses logical CSS (`ms-*`, `me-*`, `ps-*`, `pe-*`, `text-start/end`) so flipping `dir` truly mirrors composition — not just text.
- Translations as typed dictionaries in `src/i18n/{ar,en}.ts` (no i18n library needed for a landing site). A `useT()` hook returns the active dictionary.
- Arabic copy written natively (Modern Standard Arabic), not literal translations. English copy in editorial travel-magazine voice.
- Per-language typography:
  - Arabic headings: **Tajawal** (or **Rubik**) heavy weights, generous line-height
  - English headings: **Fraunces** (editorial serif with optical sizing)
  - Body (both): **Inter** for Latin, **IBM Plex Sans Arabic** for Arabic body
  - Loaded via `<link>` in `__root.tsx` head (Tailwind v4 rule).

## Visual system (torn-paper editorial collage)

Palette (defined as tokens in `src/styles.css` via `@theme`):
- `--color-sky` soft blue `#cfe2ef`
- `--color-paper` warm off-white `#f7f1e6`
- `--color-sand` `#e8d8bf`
- `--color-grass` `#7a9466`
- `--color-clay` `#b35e3c`
- `--color-ink` near-black `#1b1a17`
- `--color-muted` `#6b6357`
- Soft shadows + paper-texture overlay (subtle noise PNG generated as asset)

Reusable components:
- `<TornEdge variant="top|bottom" tone="paper|sky|sand|grass" />` — layered SVG + filtered displacement for organic ripped edges (not a simple zigzag). Uses `feTurbulence` + `feDisplacementMap` masks for tactile feel.
- `<PaperCard tilt="-2|1|3">` — white paper frame, soft shadow, optional rotation, used for postcards and testimonials.
- `<Cutout src tilt shadow>` — image with masked edges and ground-shadow, used for hero subjects and the "person on grass" moment.
- `<Section tone>` — handles background tone + torn transitions to next section.

Motion (Framer Motion, restrained):
- Hero: gentle parallax on sky + cutout
- Cards: soft hover lift + tilt
- Sections: `whileInView` fade/translate reveals
- Postcard cluster: subtle drift (loop with low amplitude)

## Technical notes

- Stack: existing TanStack Start template, Tailwind v4 tokens in `src/styles.css`, Framer Motion for animation, no backend needed (no Lovable Cloud).
- Route files: `src/routes/{index,services,packages,about,contact}.tsx` each with own `head()` metadata (unique title/description/og in both languages — English meta for crawlers, with `lang` alt).
- `src/routes/__root.tsx` provides `LanguageProvider`, sets `<html dir lang>`, loads fonts via `<link>`, renders shared `Header` / `Footer` / floating WhatsApp.
- Components in `src/components/`: `Header`, `Footer`, `LanguageSwitcher`, `WhatsAppButton`, `TornEdge`, `PaperCard`, `Cutout`, `Section`, `PostcardCluster`, `ServiceCard`, `TestimonialCard`.
- i18n: `src/i18n/{ar,en}.ts` + `src/i18n/LanguageProvider.tsx` + `useT()` hook.
- Accessibility: semantic landmarks, alt text in both languages (from dictionary), focus rings, keyboard-reachable switcher, contrast checked on paper/sky backgrounds.
- Responsive: desktop full collage, tablet preserves overlaps, mobile stacks postcards into a horizontal swipe row, hero collage simplified but still layered.
